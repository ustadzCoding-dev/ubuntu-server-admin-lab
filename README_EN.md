# 🖥️ System Administration - Ubuntu Server Lab

> Comprehensive documentation of System Administration lab with Ubuntu Server 22.04 LTS

## 📖 About This Project

This project contains comprehensive documentation from System Administration practicum covering installation and configuration of Ubuntu Server in VirtualBox virtualization environment.

### 🎯 Main Objectives
- Build a fully functional Ubuntu server
- Implement robust network configuration
- Secure remote access with SSH
- Understand virtualization and headless mode concepts

| Component | Version/Type | Description |

| 🔐 **Remote Access** | OpenSSH | Secure shell access |
